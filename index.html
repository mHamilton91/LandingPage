Tip Calculator

The tip calculator is a Javascript project that I found while exploring CodePen. This project was one of the first that projects I did using Javascript after I had gained enough experience with HTML and CSS. It is a very simple web app that once takes the bill amount, a percentage rating based on a service rating, and the number of people that the bill will be split between to calculate the proper tip.

Javascript Clock

The Javascript clock project the second project that I completed while working the 30 Days of Javascript coding challenge. The project takes in the current time and then sets the degrees based on the respective time unit, seconds, minutes, and hours.

Product Landing Page: Lumi

The product landing page is a project that I did while following the Free Code Camp projects at the end of the HTML/CSS section. The page features only HTML and CSS. I found the LUMI keyboard on Kickstarter and decided to use this as my landing page since I could easily pull information that came directly from the Kickstarter campaign to fill out the information for the product landing page.

Javascript Interactive Drum Kit

The interactive drum kit page was the first 30 Days of Javascript project that I completed. The page is set up so that the user can either press the keyboard keys marked for each part of the drum or the user can click on the buttons to play the corresponding drum sound. The original project didn't address the user being able to click on the buttons but I felt that it would allow the user more control and it added the ability to play the sounds on a smart phone or tablet.
